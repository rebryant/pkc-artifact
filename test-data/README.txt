This directory contains some very simple test formulas for the PKC projecting knowledge compiler.

Running "make test" compiles these formulas.  The results are in files
with the extension ".pkc_log"

Files:

	eg-proj.cnf
Formula used as example in paper.  Has 5 models

        majp-11.cnf
Majority formula for N=11, with a Plaisted-Greenbaum encoding.  Has
1024 models

	majt-09.cnf
Majority formula for N=9, with a Tseitin encoding.  Has 256 models
